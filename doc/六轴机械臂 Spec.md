# 六轴机械臂 Spec

## 1. Scope

本文档定义 `UAV_Robot` 项目中六轴机械臂的机械参数、关节编号、运动学参数、CAN 控制约定以及当前软件实现边界，供 `app / core / drv` 三层实现、调试和后续扩展使用。

本机械臂为六自由度步进闭环机械臂，关节驱动基于 Emm V5.0 步进闭环驱动器，通过 CAN 总线进行控制。

---

## 2. 本体参数

### 2.1 机械臂基础规格

- 自由度：6 轴
- 驱动系统：步进电机 + 高精度行星减速器
- 控制模式：FOC 闭环控制
- 通讯接口：CAN 总线
- 有效负载能力：500 g
- 最大工作半径：510 mm
- 重复定位精度：±0.03 mm
- 回零方式：机械限位 + 编码器自动回零
- 供电要求：12V DC / 10A
- 本体重量：8.3 kg（不含夹爪）
- 材质：PLA 外壳 + 金属连接件
- 安装方式：标准 M6 螺栓固定
- 安全特性：软件限位、支持软件急停
- 工作环境温度：0°C ~ 45°C
- 工作噪音：低于 55 dB

### 2.2 尺寸摘要

- 基座侧向偏置：55 mm
- 基座到肩部高度：166 mm
- 大臂长度：200 mm
- 前臂几何参数：`a3 = 56 mm`
- 腕部偏置：`d4 = 192 mm`
- 末端工具偏置：55 mm

备注：
以上参数来自当前机械臂结构图与 DH 参数，用于软件中的位置解算和工作空间近似估计。

## 4. 运动学模型

### 4.1 DH 参数

采用标准 DH 参数描述当前六轴机械臂：

| 关节 i | a (mm) | α (rad) | d (mm) | θ 范围 |
|---|---:|---:|---:|---|
| 1 | 55  | π/2  | 166 | [-180°, 160°] |
| 2 | 200 | 0    | 0   | [0°, 180°] |
| 3 | 56  | π/2  | 0   | [-80°, 83°] |
| 4 | 0   | -π/2 | 192 | [-30°, 305°] |
| 5 | 0   | π/2  | 0   | [-110°, 110°] |
| 6 | 0   | 0    | 55  | [-30°, 305°] |

### 4.2 当前软件中的 IK 使用方式

当前 `core/dev/arm.c` 中的 `arm_move_to_xyz(x, y, z)` 采用简化位置逆解：

- 使用 `J1/J2/J3` 进行末端位置求解
- `J4/J5/J6` 当前默认保持中性姿态（0°）
- 目标是先满足位置到达，再逐步扩展到完整姿态逆解

因此当前版本属于：

- 支持六关节控制
- 支持六关节同步启动
- 支持基础位置逆解
- 暂不支持任意末端姿态约束

### 5.1 关节对应电机与减速比

| 关节 | 电机型号 | 减速比 |
|---|---|---:|
| J1 | L48 | 25:1 |
| J2 | L60 | 20:1 |
| J3 | L34 | 25:1 |
| J4 | L34 | 10:1 |
| J5 | L40 | 1:1 |
| J6 | L40 | 1:1 |

### 5.2 脉冲换算

当前软件默认使用：

- 驱动基础脉冲：3200 pulses / rev

则每个关节角度对应的脉冲换算公式为：

```text
pulses_per_degree = 3200 * gear_ratio / 360
```

各关节近似值：

| 关节 | 减速比 | pulses / degree |
|---|---:|---:|
| J1 | 25 | 222.22 |
| J2 | 20 | 177.78 |
| J3 | 25 | 222.22 |
| J4 | 10 | 88.89 |
| J5 | 1  | 8.89 |
| J6 | 1  | 8.89 |

说明：

- 若驱动细分或电机步距设置变化，必须同步更新该换算关系。
- 若后续实测发现传动链存在额外传动误差，应增加每关节单独标定参数。

## 6. 通讯与协议

### 6.1 总线

- 总线类型：CAN
- 驱动协议：Emm V5.0 CAN 扩展帧协议
- 帧类型：Extended Frame
- 帧 ID 编码：

```text
can_id = (addr << 8) | packet_index
```

### 6.2 机械臂协议层职责

在项目分层中：

- `core/proto/proto_zdt_arm.c`
  负责 Emm V5.0 指令编码与回包解析
- `core/dev/arm.c`
  负责六轴机械臂语义、关节映射、同步动作、位置逆解
- `drv/dev/can_socketcan.c`
  负责 SocketCAN 收发

### 6.3 当前已实现的协议命令

机械臂协议层当前已支持：

- 使能控制
- 位置模式控制
- 速度模式控制
- 立即停止
- 多机关节同步启动
- 触发回零
- ACK / 到位回包解析

## 7. 控制语义

### 7.1 关节控制

当前机械臂的基础控制对象为六个关节：

```text
J1, J2, J3, J4, J5, J6
```

软件控制流程为：

```text
joint target (deg)
-> angle to pulses
-> proto_zdt_arm encode position
-> SocketCAN send
-> sync start
```

### 7.2 预置动作

当前 `arm_move(pose)` 已定义以下语义：

- `home`
  六关节触发回零
- `zero`
  六关节移动到 0° 姿态
- `approach`
  当前预置接近姿态
- `lift`
  当前预置抬升姿态

### 7.3 笛卡尔控制

当前 `arm_move_to_xyz(x, y, z)`：

- 输入单位：mm
- 输出：六关节目标角度
- `J1~J3` 由简化 IK 求解
- `J4~J6` 暂保持中性

这意味着当前版本适合：

- 静态抓取
- 中性姿态下的空间定位
- 六关节同步运动控制验证

不适合：

- 指定末端法向/姿态的精确装配动作
- 连续姿态规划
- 避障轨迹规划

---

## 8. 回零与安全

### 8.1 回零方式

- 机械限位 + 编码器自动回零
- 当前软件回零命令：对 J1~J6 全部下发 `trigger_home`
- 回零模式：`Nearest`

### 8.2 安全约束

软件侧当前约束包括：

- 关节软件限位检查
- 支持软件急停
- 控制前自动使能对应关节
- 支持同步启动，避免多关节分时动作导致姿态漂移

建议后续补充：

- 每关节零位偏置标定
- 每关节正方向标定
- 软限位与机械限位一致性校验
- 关节电流/堵转监控
- 回零完成状态确认
